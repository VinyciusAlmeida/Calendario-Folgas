# 📅 Meu Calendário de Folgas 2026

Este é um projeto que criei para ajudar o pessoal do trabalho a consultar as folgas de um jeito fácil pelo celular. Ele calcula sozinho quem folga em cada dia, dependendo do grupo e do setor (Produção, Manutenção ou Elétrica).

## 🌟 O que o app faz?

- **Calcula as Folgas:** Você escolhe seu grupo e o calendário pinta de **verde** os dias que você folga.
- **Mostra Feriados:** Os feriados ficam em **vermelho**, e abaixo do calendário aparece uma listinha explicando o que é cada feriado do mês.
- **Funciona no Celular:** Fiz o visual focado para quem usa smartphone, então os botões são grandes e fáceis de clicar.
- **Avisos Rápidos:** Se você tentar ir para um mês que não tem no sistema, aparece um balãozinho de aviso no pé da tela.

## 🛠️ O que eu usei para fazer?

- **HTML e CSS:** Para criar a "cara" do aplicativo e as cores.
- **JavaScript:** É o "cérebro" do projeto. Ele faz as contas das datas e troca os meses quando você clica nos botões.
- **Organização:** Separei o código em arquivos diferentes para ficar mais fácil de entender e consertar se der algum erro.

## 📂 Como o projeto está organizado?

- `index.html`: Onde fica tudo o que aparece na tela.
- `js/calculadora.js`: Onde estão as contas das folgas.
- `js/feriados.js`: Onde fica a lista de feriados.
- `js/main.js`: Onde o calendário é montado.
